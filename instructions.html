Calculator Instructions

   
         
  • 🔢 Enter the first number in the top input field.
    •      
  • 🧮 Select an operator from the dropdown:
    •      
             
    • ➕ ➖ ✖️ ➗ — Basic operations
      •        
    • 📦 `x³`, √ — Cube and square root
      •        
    • 🔼 `x^y` — Exponentiation
      •        
    • 📊 `x% of y` — Percentage calculation
      •        
    • 📐 sin, cos, tan — Trigonometric functions (in degrees)
      •      
         
  • 🔢 Enter the second number if the operation needs two values (e.g., x + y or x ^ y).
    •      
  • ⚡ Click “Compute” to see the result.
    •      
  • 📋 Click “Copy Result” if you want to copy the output to your clipboard.
    •      
  • 🧠 All calculations are automatically saved in your history.
    •      
  • 🧹 Use “Clear History” on the History page to reset saved data.
